# 買い物リスト

### 特徴：
- リストの追加機能
- スワイプ操作によるリストの削除機能
- ローカルストレージを利用したリストの保存機能
- 過去の履歴の閲覧機能
- 期限切れの履歴の自動削除機能　　
   
## 使用技術:<br>  
- **<p>開発ツール</p>**
  - React / version18.3.1
  - Vite / version5.2.12
- **<p>言語</p>**
  -  Javascript
